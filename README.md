# Django-wisdompets

### Frameworks, tecnologias, banco de dados
* Django
* SQLITE

### Descrição

Aplicação feita em Django e SQLITE
Faz cadastro de animais de estimação e vacinas, todo o controle dos dados (CRUD) é feito por meio do painel administrativo.
